# Sporttery Advisor Skill

一个用于生成竞彩观察页的 Skill。当前版本是 `V4 简化买法`：上方保留赛事摘要和当前可售赛事表，下方用四张卡片展示主票、可选加仓、小额冷门和资金矩阵。

## 使用

```bash
node sporttery_skill_v4.js
```

脚本会从竞彩网接口抓取当前足球可售赛事，并生成：

- `daily_sporttery_advisor_2026-06-14.html`
- `daily_sporttery_advisor_v4_2026-06-14.html`

## 当前规则

- 只输出三张可执行票：主票、可选加仓、小额冷门。
- 深盘赛事进入不买区，不强行纳入方案。
- 主票优先考虑让球保护，避免单纯追热门胜负。
- 页面风格采用浅灰背景、白色圆角卡片、彩色策略标题和大号预计返还。

## 提醒

本项目只做规则调试和赛前观察演示，不构成投注建议。固定奖金、可售状态和赛果以官方页面与实际出票时为准。
