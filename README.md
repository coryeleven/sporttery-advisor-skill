# 2026年世界杯体彩AI分析

基于竞彩网官方 API 实时数据，AI 自主分析赛事趋势与盘口结构，生成可执行的资金分配方案。

## 版本

当前版本：**V5.5.2**

## 功能

- 实时 API 数据抓取（竞彩网官方接口）
- AI 自主分析（让球结构、奖金分布、方向判断）
- 四套方案：稳健保底 / 价值发现 / 高危搏杀 / 矩阵对冲
- 完整风控规则：终止分支、2元法则、反幻觉、术语规范

## 使用方式

### AI 模式（推荐）

直接告诉 AI：

```text
预算 M = 50。请按"每日体彩军师"Skill 执行，抓取官方 API 赛事数据，进行 AI 分析后生成四套方案。
```

不同 AI 模型（Mimo/Claude/GPT 等）会给出不同的分析结果。

### 脚本模式

```bash
node sporttery_skill_v4.js
```

脚本会从竞彩网接口抓取当前足球可售赛事，并生成 HTML 面板。

## 界面

- 浅灰蓝配色，长时间观看舒适
- 胜平负 / 让球 / 让球胜平负 三列清晰展示
- 可售玩法用淡色标签展示
- 四宫格方案卡片
- 金色主题标题，视觉层次清晰
- 响应式布局，支持移动端

## 技术细节

- API 接口：`webapi.sporttery.cn/gateway/uniform/football/getMatchCalculatorV1.qry`
- 数据源：竞彩网官方
- 输出格式：HTML 面板 + 终止纯文本

## 风险提示

本项目只做结构化分析和页面生成，不构成投注建议。所有固定奖金、可售状态、开奖结果以官方页面和实际出票时为准。
