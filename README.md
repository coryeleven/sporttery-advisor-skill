# 每日体彩军师

基于竞彩网官方 API 实时数据，AI 自主分析赛事趋势与盘口结构，生成可执行的资金分配方案。

## 版本

当前版本：**V5.9.1**

## 功能

- 实时 API 数据抓取（竞彩网官方接口）
- AI 自主分析（让球结构、固定奖金分布、方向判断）
- 三套方案 + 最终购买建议：稳健保底 / 价值发现 / 高危搏杀 / 购买指令
- 完整风控规则：终止分支、2元法则、资金绝对平衡、反幻觉、术语规范
- 输出格式：Markdown 纯文本（默认），含「为什么这样买」和「有什么好处」

## 使用方式

直接告诉 AI：

```text
预算 M = 50。请按"每日体彩军师"Skill 执行，抓取官方 API 赛事数据，进行 AI 分析后生成三套方案。
```

不同 AI 模型（Claude / DeepSeek / GPT / Gemini / Qwen 等）会给出不同的分析结果。

## 技术细节

- API 接口：`webapi.sporttery.cn/gateway/uniform/football/getMatchCalculatorV1.qry`
- 数据源：竞彩网官方
- 默认输出格式：Markdown 纯文本
- 日期规则：业务日 N 买赛事日期 N+1 的票

## 风险提示

本项目只做结构化分析和数据报告生成，不构成投注建议。所有固定奖金、可售状态、开奖结果以官方页面和实际出票时为准。
